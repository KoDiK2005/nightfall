# visions — картинки, что мелькают при падении рассудка

Кидай сюда **PNG-файлы** (`*.png`). При запуске игра загрузит их все и будет
на доли секунды вспышками показывать поверх экрана, когда **рассудок
(РАССУДОК) низкий** — чем сильнее безумие, тем чаще и ярче. Пустая папка =
эффекта нет, всё работает как обычно.

## Требования к файлам
- Формат: **PNG, 8 бит на канал, без чересстрочности (non-interlaced)**.
  Подходят обычные PNG из любого редактора. Поддерживаются: оттенки серого,
  RGB, RGBA (прозрачность работает), индексированные (палитра) + tRNS.
- 16-битные или interlaced PNG не загрузятся (в консоль выйдет предупреждение).
  Пересохрани как обычный 8-битный PNG.
- Любой размер: слишком большие автоматически ужимаются под экран. Прозрачные
  области (альфа) остаются прозрачными — удобно для «лиц из темноты».
- До **24** картинок за раз.

## Как применить
Положил файлы → `make` → `./nightfall` (или `make run`). Ничего перегенерировать
не нужно, картинки читаются прямо во время запуска.
