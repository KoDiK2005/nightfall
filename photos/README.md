# photos — кидай сюда фотки для скримеров

Брось в эту папку любые фотографии (`.jpg`, `.jpeg`, `.png`, `.webp`, `.bmp`,
`.tif`, `.gif`) — телефонные тоже подойдут. Потом запусти конвертер:

```bash
python tools/photos2visions.py
```

Он прогонит каждое фото через хоррор-обработку (кроп под экран, черно-белый →
кроваво-красный, жёсткий контраст, виньетка, зерно) и положит готовые
`*.png` в `assets/visions/`. После этого просто запусти игру:

```bash
mingw32-make run        # Windows (MSYS2)
# или ./nightfall
```

Картинки начнут **вспышками вылазить, когда РАССУДОК падает** — чем сильнее
безумие, тем чаще и ярче. При полном рассудке их не видно.

## Полезное
- `python tools/photos2visions.py --raw` — без хоррор-фильтра, просто конверт в
  нужный формат (сохраняет прозрачность — удобно для «лиц из темноты»).
- `python tools/photos2visions.py --intensity 0.6` — слабее эффект (0..1).
- Предпросмотр в игре без ожидания безумия: задай переменную окружения
  `NIGHTFALL_SHOWVISION=0` (индекс картинки) перед запуском.

Сами исходники из `photos/` в игру не попадают и в git не коммитятся —
коммить только то, что окажется в `assets/visions/`, если захочешь.
