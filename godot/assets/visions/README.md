# visions — картинки, что мелькают при падении рассудка

Кидай сюда **PNG-файлы** (`*.png`). При запуске игра загрузит их все и будет
на доли секунды вспышками показывать поверх экрана, когда **рассудок
(РАССУДОК) низкий** — чем сильнее безумие, тем чаще и ярче. Пустая папка =
эффекта нет, всё работает как обычно.

## Требования к файлам
- Формат: **PNG**, любой (Godot грузит его через встроенный `Image.load_from_file` —
  никакого самодельного декодера, как в C-версии, тут не нужно).
- Любой размер: слишком большие автоматически ужимаются под экран. Прозрачные
  области (альфа) остаются прозрачными — удобно для «лиц из темноты».
- До **24** картинок за раз.

## Как применить
Положил файлы → перезапустил игру (`godot4 .`). Ничего перегенерировать не
нужно, картинки читаются прямо во время запуска, как и в C-версии.

## Проверка без ожидания низкого рассудка
`NIGHTFALL_SHOWVISION=i` (порт одноимённого dev-хука из C) сразу показывает
картинку с индексом `i` и держит её на экране, а не мелькает долей секунды —
удобно для скриншотов/отладки.
